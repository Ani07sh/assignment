# Advance English Classes — Website

A professional website built for a local English coaching 
center in Asansol, West Bengal.

## About
Advance English Classes specializes in Spoken English, 
Grammar and Textual study for students of Class 8 to 12 
(WBBSE, WBCHSE and CBSE boards).

## Pages / Sections
- Hero section with stats
- Subjects (Spoken, Grammar, Textual)
- Classes (Class 8–12)
- Results & student success stories
- Faculty section
- Fee structure
- Free demo class booking form
- Contact information

## Built With
- HTML5
- CSS3
- JavaScript (vanilla)
